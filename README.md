# gh-genissues

Populate and generate a list of issues to do any kind of project, especially when you have a document with a bunch of expectations, you don't want click away your day
by creating multiple issues and setting them. Say no more! I'd say this project is for you.


## ðŸ“¦ Installation

1. Install the `gh` CLI - see the [installation](https://github.com/cli/cli#installation)

   _Installation requires a minimum version (2.0.0) of the the GitHub CLI that supports extensions._

2. Install this extension:

   ```sh
   gh extension install igorcosta/gh-genissues
   ```

<details>
   <summary>Installing Manually</summary>

> If you want to install this extension **manually**, follow these steps:

1. Clone the repo

   ```bash
   # git
   git clone https://github.com/igorcosta/gh-genissues

   # GitHub CLI
   gh repo clone igorcosta/gh-genissues
   ```

2. Cd into it

   ```bash
   cd gh-genissues
   ```

3. Install it locally
   ```bash
   gh extension install .
   ```
